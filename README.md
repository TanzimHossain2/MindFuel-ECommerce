# MindFuel-ECommerce

## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/TanzimHossain2/MindFuel-ECommerce.git
   ```
2. Change directory
   ```sh
   cd MindFuel-ECommerce
   ```
3. Install NPM packages
   ```sh
    yarn install
   ```
4. Start the project
   ```sh
     yarn dev
   ```
5. Open [http://localhost:8000](http://localhost:8000) to view it in the browser.
